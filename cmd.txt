cd текущая папка программа переключающая текущую папку 
cd ~ переходит в домашнюю папку
cd тоже переходит в домашнюю папку
cd . перейти в текущую папку
. ./ текущая папка
.. предыдущая папка
cd .. переход в предыдущую папку
pwd полный путь до текущей деректории
ls показать содержимое по умолчанию в текущей папке
ls -l показать содержимое списка показывает права доступа к файлам и папкам
git программа контроля версий
git clone клонирование удалённого репозитория
git add . добавить содержимое текущей папки в систему контроля версий
git status показывает состояние версий
git commit сохраняет изменения файлов
git push заливает изменения в репозиторий 
Репозиторий папка с отслежевающим кодом 
touch программа чтобы тронуть файл обновляет последние изменения если не существует то создаётся новый
mkdir создать папку 
sudo 
sudo su даёт админские права
sudo apt-get install имя пакета который хочешь установить
